This project is about predicting the price of books.
## Data of books used:
* Title           
* Author          
* Edition         
* Reviews         
* Ratings         
* Synopsis        
* Genre of the book
* Category of the book
* Price (target column)

## Link to hackathon and dataset:
link to the hackathon: https://www.machinehack.com/hackathons/5e8327d452c028cd80a0bd9a
